Курс по теме Кластерный и факторный анализ

Обзор курса

В данном курсе реализуется объяснительно-иллюстративное обучение.

Основные функции преподавателя: предъявление информации и контроль усвоения.

Принципы организации для этой модели обучения:

  • образование в открытой коммуникативной среде;

Реализуются следующие педагогические технологии:

  • проблемное обучение;
  • модельное обучение.

Учебно-методический комплекс включает в себя:
  • рабочую программу дисциплины;
  • конспекты лекций, медиаматериалы в виде презентаций по темам;
  • материалы для практических занятий: задания для выполнения лабораторных работ на компьютере;
  • материалы для организации самостоятельной работы студентов;
  • контрольно-измерительные материалы;
  • список литературы (в том числе интернет-ресурсов);
  • глоссарий;
  • дополнительные материалы: учебники (в том числе электронные), монографии, статьи, медиаматериалы, база образовательных интернет-ресурсов по Data Minning.
